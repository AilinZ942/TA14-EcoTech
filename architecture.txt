Layer Summary
-------------
1. Presentation layer:
   Vue views in `src/views`

2. Routing layer:
   Centralized route definitions in `src/router/index.js`

3. Map integration layer:
   `src/lib/mapApi.js` and `src/lib/ewasteMapModel.js`

4. Backend/API layer:
   Azure-hosted endpoints plus local bundled data assets

5. Data asset layer:
   CSV, GeoJSON, and geometry files used by the map views


Current important contracts observed in repo
--------------------------------------------
- AI chat:
  `api.getPerson()` -> `GET /api/GetPerson`

- Disposal map:
  `searchMapFacilities(payload)` -> unified `{ items, meta }` response

- Disposal data tracks:
  Azure API, local CSV, optional legacy GeoJSON fallback
